# 学科分类与代码参考（GB/T 13745-2009）

本标准共设 **62 个一级学科**、**676 个二级学科**、**2382 个三级学科**。以下为精简结构，用于知识框架激活。

---

### 150 化学

### 160 天文学

### 180 生物学

## 国际补充：常见交叉与新兴学科

- **数据科学**：统计学 + 计算机 + 数学
- **认知科学**：心理学 + 神经科学 + 语言学 + 哲学 + 人工智能
- **环境科学**：地球科学 + 化学 + 生物学 + 社会学
- **生物信息学**：生物学 + 计算机 + 数学
- **计算社会科学**：社会学 + 计算机 + 统计学
- **科技伦理**：哲学 + 法学 + 科学技术
- **数字人文**：文学 + 历史 + 计算机 + 图书馆学
