# 易经占卜 (Yijing Divination)

当面临人生抉择或需要指引时，让易经的智慧为你指明方向。

## ⚠️ 重要：节省 Token

**不要读取 `scripts/64_hexagrams.json`。** 该文件体量大，仅用于脚本内部；占卜与解读应基于你对易经卦象、卦辞、爻辞的已有知识完成，读取该 JSON 会浪费大量 token 且无必要。

## 使用方法

1. **执行脚本**：运行 `scripts/divinate.py`，由脚本生成卦象（主卦、变卦等）。
2. **根据输出解读**：直接依据脚本打印的结果进行解读，结合卦辞、爻辞和变爻给出指引。

## 占卜方法

### 金钱法（唯一方法）

1. **静心凝神**：专注于你的问题，清空杂念
2. **投掷硬币**：同时投掷三枚硬币，记录结果
   - 正面（阳）= 3点
   - 反面（阴）= 2点
3. **计算点数**：三枚硬币点数之和
   - 6点（三反）：老阴（变爻）→ 阴爻
   - 7点（二反一正）：少阳（不变）→ 阳爻
   - 8点（二正一反）：少阴（不变）→ 阴爻
   - 9点（三正）：老阳（变爻）→ 阳爻
4. **记录爻象**：从下到上共六次，形成完整卦象
5. **确定卦名**：根据爻象使用周易标准卦序确定卦名

## 脚本使用

### Python 脚本

```bash
python scripts/divinate.py
```

## 卦象解读流程

1. **主卦**：原始生成的卦象
2. **变爻分析**：老阴变阳、老阳变阴
3. **变卦**：变爻变化后形成的卦象
4. **综合解读**：
   - 主卦：当前状况
   - 变卦：未来趋势
   - 变爻：关键转折点
   - 卦辞：整体指导
   - 爻辞：具体建议

指导：机会良好，但需要关注执行细节，积极主动地面对新挑战。
